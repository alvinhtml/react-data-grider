# react-lister

react data list

## Development

- Install npm packages

    ```
  npm install
    ```

- Launch dev server

    ```
  npm run start
    ```

- Lint

    ```
  npm run lint
    ```

- Build

    ```
  npm run build
    ```



## Install react-miniui

```bash
npm install react-lister --save
```

## Usage

```js
import Lister, {column} from 'react-lister';
```
